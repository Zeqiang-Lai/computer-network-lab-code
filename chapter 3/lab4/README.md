## 输出格式

`|`表示或，输出其中的一个。

发送：

```
------------------ 发送 --------------------------------
Ack_expected=1,  Next_frame_to_send=1,  Frame_expected=1
待发送帧: [Num=1] [Ack=1] [Data="00000101010"]
模拟: 正确发送 | 传输出错  | 丢失
```

接受：

```
------------------ 接受 --------------------------------
收到帧: [Num=1] [Ack=1] [Data="00000101010"]
Frame_expected=1,  Ack_expected=1
CRC校验成功  | CRC校验失败
```

超时：

```
------------------ 超时 --------------------------------
Ack_expected=1,  Next_frame_to_send=1,  Frame_expected=1
待发送帧: [Num=1] [Ack=1] [Data="00000101010"]
```
